# Projeto da disciplina de Desenvolvimento Frontend para Web

A plataforma será um sistema web completo que oferece às ONGs uma presença digital profissional e funcional.

## Entregas

A disciplina espera a entrega de quatro práticas que serão colocadas em branchs separadas. Cada entrega seguirá o seguinte padrão de nomeação de branch:

- a i-ésima entrega estará na branch `entrega-$i`. Exemplo entrega II -> branch `entrega-II`
